Блок навигации

  • Ширина 15em
  • Высота определяется контентом
  • Должен присутсвовать цвет фона
  • Располагается слева
  • Блок навигации наполниет контентом достаточным, чтобы по высоте он начинал захватывать вторую статью, как показано в данном примере

Статья 1

  • Размер блока:
  • Занимает всю доступную ширину
  • Высота по размеру контента
  • Должен присутствовать цвет фона
  • Во всех "статьях" сделвйте отступ снизу от соседей равным 0.2em, при помощи свойства margin-bottom

Статья 2

  • Размер блока:
  • Занимает всю дотупную ширину
  • Высота по размеру контента
  • Должен присутствовать цвет фона
  • Во всех "статьях" сделвйте отступ снизу от соседей равным 0.2em, при помощи свойства margin-bottom. Картинка вставлена при помощи тега, плавание организованно по правому краю внутри данной статьи для этой картинки
Нижний блок, подвал. Занимает всю доступную ширину. Высота определяется помещенным внутрь контентом.
